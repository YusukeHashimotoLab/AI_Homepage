Ë™øÂíåÊåØÂãïÂ≠ê„É¢„Éá„É´

ÁµêÊô∂‰∏≠„ÅÆÂéüÂ≠ê„ÅØÂπ≥Ë°°‰ΩçÁΩÆ$\mathbf{R}_I^0$„ÅÆÂë®„Çä„ÅßÊåØÂãï„Åó„Åæ„Åô„ÄÇÂ§â‰Ωç„Çí$\mathbf{u}_I$„Å®„Åô„Çã„Å®Ôºö $$ \mathbf{R}_I = \mathbf{R}_I^0 + \mathbf{u}_I $$ „Éù„ÉÜ„É≥„Ç∑„É£„É´„Ç®„Éç„É´„ÇÆ„Éº„ÇíTaylorÂ±ïÈñãÔºàË™øÂíåËøë‰ººÔºâÔºö $$ U = U_0 + \sum_I \frac{\partial U}{\partial \mathbf{u}_I}\Bigg|_0 \mathbf{u}_I + \frac{1}{2}\sum_{I,J} \mathbf{u}_I \cdot \frac{\partial^2 U}{\partial \mathbf{u}_I \partial \mathbf{u}_J}\Bigg|_0 \cdot \mathbf{u}_J + O(\mathbf{u}^3) $$ Âπ≥Ë°°‰ΩçÁΩÆ„Åß„ÅØÁ¨¨1È†Ö„Åå„Çº„É≠„Å™„ÅÆ„ÅßÔºö $$ U \approx U_0 + \frac{1}{2}\sum_{I,J} \mathbf{u}_I \cdot \mathbf{\Phi}_{IJ} \cdot \mathbf{u}_J $$ $\mathbf{\Phi}_{IJ}$„ÅØÂäõÂÆöÊï∞Ë°åÂàóÔºàforce constant matrixÔºâÔºö $$ \Phi_{IJ}^{\alpha\beta} = \frac{\partial^2 U}{\partial u_I^\alpha \partial u_J^\beta}\Bigg|_0 $$

ÈÅãÂãïÊñπÁ®ãÂºè

ÂéüÂ≠ê$I$„ÅÆÈÅãÂãïÊñπÁ®ãÂºèÔºö $$ M_I \frac{d^2 u_I^\alpha}{dt^2} = -\sum_{J,\beta} \Phi_{IJ}^{\alpha\beta} u_J^\beta $$ Âπ≥Èù¢Ê≥¢Ëß£„Çí‰ªÆÂÆöÔºö $$ u_I^\alpha = \frac{1}{\sqrt{M_I}} e_I^\alpha(\mathbf{k}) e^{i(\mathbf{k}\cdot\mathbf{R}_I - \omega t)} $$ ÂãïÂäõÂ≠¶Ë°åÂàóÔºàdynamical matrixÔºâ„ÇíÂ∞éÂÖ•Ôºö $$ D_{IJ}^{\alpha\beta}(\mathbf{k}) = \frac{1}{\sqrt{M_I M_J}} \Phi_{IJ}^{\alpha\beta} e^{i\mathbf{k}\cdot(\mathbf{R}_I - \mathbf{R}_J)} $$

$$ \sum_{J,\beta} D_{IJ}^{\alpha\beta}(\mathbf{k}) e_J^\beta(\mathbf{k}) = \omega^2(\mathbf{k}) e_I^\alpha(\mathbf{k}) $$ - $\omega(\mathbf{k})$: „Éï„Ç©„Éé„É≥„ÅÆËßíÊåØÂãïÊï∞Ôºà„Éï„Ç©„Éé„É≥ÂàÜÊï£Ôºâ - $\mathbf{e}(\mathbf{k})$: „Éï„Ç©„Éé„É≥„ÅÆÂõ∫Êúâ„Éô„ÇØ„Éà„É´ÔºàÂÅèÊ•µ„Éô„ÇØ„Éà„É´Ôºâ „Éï„Ç©„Éé„É≥„Éê„É≥„Éâ„ÅÆÊï∞: $3N_{\text{atom}}$Êú¨ÔºàÂçò‰ΩçËÉûÂÜÖÂéüÂ≠êÊï∞$N_{\text{atom}}$„Å´ÂØæ„Åó„Å¶Ôºâ ---

from ase.build import bulk
from gpaw import GPAW, PW
from phonopy import Phonopy
from phonopy.structure.atoms import PhonopyAtoms
import numpy as np

def calculate_phonon(symbol='Si', a=5.43, dim=(2,2,2)):
    """
    ÂÆåÂÖ®Ëá™Âãï„Éï„Ç©„Éé„É≥Ë®àÁÆó
    """
    # 1. Âü∫Â∫ïÁä∂ÊÖãË®àÁÆó
    print("Step 1: Ground state calculation...")
    atoms = bulk(symbol, 'diamond', a=a)
    calc = GPAW(mode=PW(400), xc='PBE', kpts=(8,8,8), txt=None)
    atoms.calc = calc
    atoms.get_potential_energy()

    # 2. Phonopy„Çª„ÉÉ„Éà„Ç¢„ÉÉ„Éó
    print("Step 2: Generate displacements...")
    cell = PhonopyAtoms(symbols=[symbol]*len(atoms),
                       cell=atoms.cell,
                       scaled_positions=atoms.get_scaled_positions())

    phonon = Phonopy(cell, np.diag(dim))
    phonon.generate_displacements(distance=0.01)

    # 3. Âäõ„ÅÆË®àÁÆó
    print(f"Step 3: Calculate forces for {len(phonon.supercells_with_displacements)} supercells...")
    set_of_forces = []

    for i, scell in enumerate(phonon.supercells_with_displacements):
        supercell = bulk(symbol, 'diamond', a=a).repeat(dim)
        supercell.positions = scell.positions
        supercell.calc = calc
        forces = supercell.get_forces()
        drift_force = forces.sum(axis=0)
        for force in forces:
            force -= drift_force / len(forces)  # „Éâ„É™„Éï„ÉàË£úÊ≠£
        set_of_forces.append(forces)

    # 4. „Éï„Ç©„Éé„É≥Ë®àÁÆó
    print("Step 4: Phonon calculation...")
    phonon.produce_force_constants(forces=set_of_forces)

    # „Éê„É≥„ÉâÊßãÈÄ†
    path = [[[0, 0, 0], [0.5, 0, 0.5], [0.625, 0.25, 0.625],
             [0.375, 0.375, 0.75], [0, 0, 0], [0.5, 0.5, 0.5]]]
    labels = ["$\\Gamma$", "X", "U", "K", "$\\Gamma$", "L"]

    phonon.auto_band_structure(plot=True, labels=labels, filename=f'{symbol}_band.png')

    # DOS
    phonon.auto_total_dos(plot=True, filename=f'{symbol}_dos.png')

    print("Done!")
    return phonon

4.4 ÁÜ±ÂäõÂ≠¶ÁâπÊÄß„ÅÆË®àÁÆó

Ëá™Áî±„Ç®„Éç„É´„ÇÆ„Éº

HelmholtzËá™Áî±„Ç®„Éç„É´„ÇÆ„ÉºÔºàNVTÔºâ: $$ F(T) = U_0 + k_B T \sum_{\mathbf{q},j} \ln\left[2\sinh\left(\frac{\hbar\omega_{\mathbf{q}j}}{2k_B T}\right)\right] $$ - $U_0$: „Çº„É≠ÁÇπ„Ç®„Éç„É´„ÇÆ„Éº - $\omega_{\mathbf{q}j}$: Ê≥¢Êï∞$\mathbf{q}$„ÄÅ„Éê„É≥„Éâ$j$„ÅÆ„Éï„Ç©„Éé„É≥Âë®Ê≥¢Êï∞

ÂÜÖÈÉ®„Ç®„Éç„É´„ÇÆ„Éº

$$ U(T) = U_0 + \sum_{\mathbf{q},j} \hbar\omega_{\mathbf{q}j} \left[\frac{1}{2} + n_B(\omega_{\mathbf{q}j}, T)\right] $$ $n_B(\omega, T)$„ÅØBose-EinsteinÂàÜÂ∏ÉÈñ¢Êï∞Ôºö $$ n_B(\omega, T) = \frac{1}{e^{\hbar\omega/(k_B T)} - 1} $$

„Ç®„É≥„Éà„É≠„Éî„Éº

$$ S(T) = -\left(\frac{\partial F}{\partial T}\right)_V = k_B \sum_{\mathbf{q},j} \left[\frac{\hbar\omega_{\mathbf{q}j}}{k_B T} n_B(\omega_{\mathbf{q}j}, T) - \ln(1 - e^{-\hbar\omega_{\mathbf{q}j}/(k_B T)})\right] $$

ÊØîÁÜ±ÔºàÂÆöÁ©çÔºâ

$$ C_V(T) = \left(\frac{\partial U}{\partial T}\right)_V = k_B \sum_{\mathbf{q},j} \left(\frac{\hbar\omega_{\mathbf{q}j}}{k_B T}\right)^2 \frac{e^{\hbar\omega_{\mathbf{q}j}/(k_B T)}}{(e^{\hbar\omega_{\mathbf{q}j}/(k_B T)} - 1)^2} $$
