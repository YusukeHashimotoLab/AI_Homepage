ÂéüÁêÜ: „Ç´„Éº„Éç„É´Ê≥ïÔºàGaussian ProcessÔºâ $$ E_{\text{GAP}}(\mathbf{R}) = \sum_{i=1}^N \alpha_i K(\mathbf{R}, \mathbf{R}_i) $$ - $K$: „Ç´„Éº„Éç„É´Èñ¢Êï∞ÔºàÈ°û‰ººÂ∫¶„ÇíÊ∏¨„ÇãÔºâ - $\mathbf{R}_i$: Ë®ìÁ∑¥„Éá„Éº„Çø„ÅÆÂéüÂ≠êÈÖçÁΩÆ - $\alpha_i$: Ë®ìÁ∑¥„Éë„É©„É°„Éº„Çø ÁâπÂæ¥: - ‚úÖ ‰∏çÁ¢∫ÂÆüÊÄßÊé®ÂÆö„ÅåÂèØËÉΩÔºàActive Learning„Å´ÊúâÂà©Ôºâ - ‚úÖ Â∞ë„Å™„ÅÑ„Éá„Éº„Çø„ÅßÂ≠¶ÁøíÂèØËÉΩ - ‚ùå Ë®ìÁ∑¥„Éá„Éº„ÇøÊï∞„Å´ÊØî‰æã„Åó„Å¶Ë®àÁÆó„Ç≥„Çπ„ÉàÂ¢óÂä†

2. Neural Network Potential (NNP)

Behler-ParrinelloÂûã: ÂêÑÂéüÂ≠ê„ÅÆ„É≠„Éº„Ç´„É´Áí∞Â¢É„ÇíË®òËø∞Â≠êÂåñ $$ E_{\text{NNP}} = \sum_{i=1}^{N_{\text{atoms}}} E_i^{\text{NN}}(\{\mathbf{G}_i\}) $$ - $E_i^{\text{NN}}$: ÂéüÂ≠ê$i$„ÅÆ„Éã„É•„Éº„É©„É´„Éç„ÉÉ„Éà„ÉØ„Éº„ÇØ„Ç®„Éç„É´„ÇÆ„Éº - $\mathbf{G}_i$: ÂØæÁß∞Èñ¢Êï∞ÔºàSymmetry FunctionsÔºâ„ÄÅÂéüÂ≠ê$i$„ÅÆÂë®Âõ≤Áí∞Â¢É„ÇíË®òËø∞ ÂØæÁß∞Èñ¢Êï∞„ÅÆ‰æãÔºàÂãïÂæÑÊàêÂàÜÔºâ: $$ G_i^{\text{rad}} = \sum_{j \neq i} e^{-\eta(r_{ij} - R_s)^2} f_c(r_{ij}) $$ - $r_{ij}$: ÂéüÂ≠êÈñìË∑ùÈõ¢ - $f_c(r)$: „Ç´„ÉÉ„Éà„Ç™„ÉïÈñ¢Êï∞Ôºà‰∏ÄÂÆöË∑ùÈõ¢‰ª•ÈÅ†„ÇíÁÑ°Ë¶ñÔºâ ÁâπÂæ¥: - ‚úÖ Â§ßË¶èÊ®°Á≥ª„Åß„ÇÇÈ´òÈÄü - ‚úÖ Ë®ìÁ∑¥„Éá„Éº„ÇøÊï∞„ÅåÂ¢ó„Åà„Å¶„ÇÇË®àÁÆó„Ç≥„Çπ„Éà‰∏ÄÂÆö - ‚ùå ‰∏çÁ¢∫ÂÆüÊÄßÊé®ÂÆö„ÅåÂõ∞Èõ£

3. Message Passing Neural Network (MPNN)

„Ç∞„É©„Éï„Éã„É•„Éº„É©„É´„Éç„ÉÉ„Éà„ÉØ„Éº„ÇØÔºàGNNÔºâ„ÅÆ‰∏ÄÁ®ÆÔºö $$ \mathbf{h}_i^{(k+1)} = \text{Update}\left(\mathbf{h}_i^{(k)}, \sum_{j \in \mathcal{N}(i)} \text{Message}(\mathbf{h}_i^{(k)}, \mathbf{h}_j^{(k)}, \mathbf{e}_{ij})\right) $$ - $\mathbf{h}_i^{(k)}$: ÂéüÂ≠ê$i$„ÅÆ$k$Â±§ÁõÆ„ÅÆÈö†„ÇåÁä∂ÊÖã - $\mathcal{N}(i)$: ÂéüÂ≠ê$i$„ÅÆËøëÂÇçÂéüÂ≠ê - $\mathbf{e}_{ij}$: ÁµêÂêàÊÉÖÂ†±ÔºàË∑ùÈõ¢„ÄÅËßíÂ∫¶Ôºâ ‰ª£Ë°®ÁöÑ„É¢„Éá„É´: SchNet„ÄÅDimeNet„ÄÅGemNet„ÄÅMACE ÁâπÂæ¥: - ‚úÖ ÂõûËª¢„Éª‰∏¶ÈÄ≤‰∏çÂ§âÊÄß„ÇíËá™ÁÑ∂„Å´ÂÆüÁèæ - ‚úÖ Èï∑Ë∑ùÈõ¢Áõ∏‰∫í‰ΩúÁî®„ÇíÂäπÁéáÁöÑ„Å´Â≠¶Áøí - ‚úÖ ÊúÄÊñ∞„ÅÆÈ´òÁ≤æÂ∫¶„É¢„Éá„É´

def generate_training_data(n_samples=50):
    """
    Ê∞¥ÂàÜÂ≠ê„ÅÆÊßò„ÄÖ„Å™ÈÖçÁΩÆ„ÅßDFTË®àÁÆó
    """
    from ase.md.velocitydistribution import MaxwellBoltzmannDistribution
    from ase.md.verlet import VelocityVerlet
    from ase import units

    h2o = molecule('H2O')
    h2o.center(vacuum=5.0)

    # DFTË®àÁÆóÊ©ü

    h2o.calc = calc

    # ÂàùÊúüÈÄüÂ∫¶

    images = []

        atoms_copy.get_potential_energy()  # DFTË®àÁÆóÂÆüË°å
        atoms_copy.get_forces()
        images.append(atoms_copy)
        print(f"Sample {i+1}/{n_samples} collected")

    return images
