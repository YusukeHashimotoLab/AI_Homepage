2025年1月 論文発表

APL Machine Learning誌に論文発表

橋本特任准教授らの研究チームが、AIを用いた「材料マップ」の作成に関する論文を国際誌APL Machine Learningに発表しました。 この研究では、機械学習を用いて膨大な材料データベースから有用な情報を迅速に抽出し、材料発見を加速する手法を実証しています。

論文情報: Y. Hashimoto, X. Jia, H. Li, T. Tomai, "A materials map integrating experimental and computational data via graph-based machine learning for enhanced materials discovery", APL Machine Learning 3, 036104 (2025)
